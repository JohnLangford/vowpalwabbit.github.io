Powerful new ML tools from cutting-edge research

Vowpal Wabbit (VW) is a mature open-source machine learning library of ultra-fast tools that solve disparate problems--including extreme classification--within microseconds. Scalable and easy to use with flexible input format, VW's intrinsic speed efficiently manages online machine learning for personalization, ranking, prediction, and decision services, as well as advanced research.

Explore VW's distinctive implementation of contextual bandits, learn to search, extreme classification, online learning, and more.
