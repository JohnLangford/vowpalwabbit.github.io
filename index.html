Powerful ML tools from cutting-edge research

Vowpal Wabbit (VW) is the mature open-source online machine learning library that learns solution to disparate problems within milliseconds. Scalable and easy to use with a flexible input format, VW's online machine learning supports personalization, interactive learning, prediction and classification, as well as advanced research.

Explore VW's contextual bandits, learn to search, extreme classification, online learning, and more.
